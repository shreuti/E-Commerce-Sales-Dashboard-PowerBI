# E-Commerce Sales Dashboard using Power BI

## 📌 Project Overview
An interactive Power BI dashboard built using the Sample Superstore dataset to analyze sales performance and business insights.

## ✨ Features
- Total Sales, Profit, Orders & Quantity KPI Cards
- Sales by Category
- Sales by Region
- Sales Trend Analysis
- Sales by Segment
- Top 10 Products by Sales
- Top 10 Customers
- Top 10 Sub-Categories
- Interactive Region & Category Slicers

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Sample Superstore Dataset
